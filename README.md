# 🚀 AI Resume Builder

An AI-powered Resume Builder designed to help users create professional, ATS-friendly resumes with ease. The application provides multiple modern templates, real-time preview, customization options, and export functionality for a seamless resume-building experience.

## ✨ Features

* 📄 Multiple Resume Templates
* 🤖 ATS Score Analysis
* 👀 Real-Time Resume Preview
* 📥 Export Resume as PDF
* 📄 Export Resume as DOCX
* 🖼️ Profile Photo Upload
* 🎨 Custom Color Themes
* 🔤 Font Selection
* 🔗 Social Links Integration

  * LinkedIn
  * GitHub
  * Portfolio
  * LeetCode
  * HackerRank
* 📱 Fully Responsive Design
* ⚡ Fast and User-Friendly Interface

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)
